# Quiz

The Quiz Application is an interactive software program designed to test a user's knowledge through Multiple Choice Questions (MCQs). It presents a series of questions along with four answer options, allowing the user to select the correct answer. After answering all questions, the application calculates and displays the final score.

This project is developed using Java Swing for the graphical user interface (GUI) and Event Handling for user interaction. It provides a simple and user-friendly way to conduct quizzes and evaluate performance.

FEATURES:-
1. Multiple Choice Questions (MCQs)
2. Single-answer selection using Radio Buttons
3. Next and Submit buttons
4. Automatic score calculation
5. Final result display
6. Easy-to-use graphical interface

TECHNOLOGIES USED:-
1. Java
2. Java Swing (GUI)
3. Event Handling (ActionListener)

OBJECTIVES:-
1. To develop an interactive quiz system.
2. To evaluate users' knowledge through MCQ-based questions.
3. To provide instant feedback and score calculation.
4. To improve understanding of Java GUI programming and event handling.

APPLICATIONS:-
1. Educational institutions
2. Online tests and assessments
3. Self-learning and practice quizzes
4. Training and evaluation systems
